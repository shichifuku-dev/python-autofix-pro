# Privacy Policy — Python Autofix Pro

Last updated: 2026-02-01

Python Autofix Pro is a GitHub App that runs Ruff formatting and lint autofixes on pull requests and reports results as GitHub Checks.

## Data we access
When installed on a repository, the app may access:
- Repository metadata (repository name, IDs)
- Pull request metadata (PR number, commit SHA)
- Pull request file list and changed files (to run Ruff)
- Check run results and statuses
- Issue data (only if you use the in-repo “settings issue” feature)

## What we store
We store only what is necessary to operate the app:
- Installation ID and repository ID
- Minimal configuration values (e.g., enableUnsafeFixes)
- Operational logs for debugging and reliability

We do not sell personal data.

## Billing (paid plans only)
For paid plans, we may receive GitHub Marketplace webhook events for:
- subscription created / updated / cancelled

These events are used only for plan verification and access control.
No billing or payment card data is processed by us.

## Data sharing
We do not share your repository content with third parties for advertising.
We may use infrastructure providers to host the service and logs, strictly for operating the app.

## Security
We follow standard security practices for GitHub Apps:
- Short-lived installation tokens
- Least-privilege permissions
- Secure storage of secrets

## Your choices
- Uninstall the app to stop all access.
- Remove the settings issue to remove repository-level configuration.
